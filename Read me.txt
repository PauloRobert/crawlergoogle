For Proper execution of this programme,please Install the following libraries in your OS:
Python,Scrapy==1.6,autopep8,pylint,selenium,scrapy-selenium,scrapy-xlsx

You can install this in following ways:
1)Python = Download and install from python.org 
 2)Libraries: Run the following commands one by one in CMD:
	pip install scrapy==1.6
	pip install autopep8
	pip install pylint
	pip install selenium
	pip install scrapy-selenium
	pip install scrapy-xlsx

Recommendation:
	1)Its better to have high speed internet connection before data extraction
	2)Close other browsers or windows before executing and leave the pc as it is till it completes data extraction.
